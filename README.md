# Todo api

This is a basic todo api with authentication. Built with express.js.

## Key Features

- Authentication
  - Login [Public]
  - SignUp [Public]
  - Tokens [User]
- Task
  - View task[Public]
  - Create task[User]
  - Update task [User]

## Run the server

- cd /src
- node server.js
